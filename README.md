# WordPress-ImgBed-Integration
将图片同步到 Cloudflare ImgBed，提供三种本地文件保留策略，完美兼容wordpress子比主题。

本插件根据cloudflare imgbed官方上传API开发，仅针对于上传制作

插件安装后配置页面路径

访问你的站点 https://www.domin.com/wp-admin/ 👉 设置 👉 Cloudflare ImgBed 设置选项 

按照要求配置即可~

如何安装？

在你网站plugin路径下新建 cloudflare-imagebed 文件夹 👉 在cloudflare-imagebed 文件夹 中新建一个 cf_imgbed.php 文件

填入本项目中cf_imgbed.php的代码即可
